하이퍼링크 관련 태그 : a(anchor)

a태그는 현재 HTML문서와 다른 HTML문서를 연결해서 페이지 이동을 할 수 있게 하는 기능을 제공한다.
-> 이 떄, a태그로 감싸진 문자열 == 히이퍼링크

* 하이퍼링크는 웹(HTML) 문서가 다른 문서와 구분될 수 있는 핵심적인 기능이다.
  텍스트 또는 이미지를 클릭해서 페이지 이동
  + 페이지 내에서 이동도 가능

a태그를 이용한 하이퍼링크 구현

  • 01_글자 관련 태그 target
  • 02_목록 관련 태그 blank
  • 03_표 관련 태그 parent
  • 06_영역 관련 태그 self
  • 07_이미지 관련 태그 top
  • 네이버
